🧮 剂量计算器

1 · 肌酐清除率 CrCl（Cockcroft-Gault）

用于经肾排泄药物的减量判断。用到年龄、体重、性别、血肌酐——这是性别真正影响剂量的主要途径。
输入后自动计算
CrCl = (140−年龄) × 体重(kg) × (女 ×0.85) / (72 × SCr[mg/dL])　肥胖者宜用校正体重

2 · 儿童按体重折算

儿科多数药物按 mg/kg 给药。输入体重与单位剂量得出每次/每日量。
输入后自动计算

3 · 泵速换算 μg/kg/min ↔ mL/h

多巴胺、间羟胺、利多卡因等持续泵入。输入体重、目标速率与药液浓度得出输液泵设定。
输入后自动计算
mL/h = 速率(μg/kg/min) × 体重 × 60 ÷ 浓度(μg/mL)；浓度 = 药量(mg)×1000 ÷ 总量(mL)

4 · 体表面积 BSA（Mosteller）

部分药物按 BSA(m²)给药。
输入后自动计算
BSA(m²) = √(身高cm × 体重kg ÷ 3600)

5 · 成人量 → 小儿量 折算（"酌减"落地）

把含糊的"酌减"变成可算的估值：体重法 + 体表面积法同时给出（单位与你输入的成人量一致）。身高填了用更准的 Mosteller 估 BSA，不填则按体重估算。
输入成人剂量与小儿体重后自动计算
体重法 = 成人量 × 小儿体重 ÷ 60　|　体表面积法 = 成人量 × 小儿BSA ÷ 1.73
仅适用于：儿童本就可用 · 非窄治疗窗(地高辛/茶碱等靠血药监测) · 不含雄黄/乌头碱/朱砂等不可线性缩放的毒性成分
计算结果仅供辅助，须结合患者实际情况与说明书核对；危重/儿科用药请双人核对。

📋 制度与目录速览

① 医保目录（能不能报销）

全称《国家基本医疗保险、工伤保险和生育保险药品目录》，决定药品能否报销及比例。
甲类按比例全额纳入报销；乙类需个人先自付一定比例再报销。现行2025版，共约3253种(西药1857、中成药1396)，每年调整一次，由各省统一执行、不得自行增删品种与甲乙分类。
查询入口：国家医保局微信公众号 →"国家医保药品目录查询"。

② 国家基本药物目录（基药）

"保基本"清单，与医保目录是两份不同的名单。
现行2018年版（685种），强调临床必需、安全有效、价格合理、保障供应。实行"1+X"配备模式（"1"=基药，"X"=非基药）：基层及低级别医院基药配备/使用比例要求更高——这正是"不同级别医院常用药不同"的制度根源之一。基药≠医保目录，勿混。

③ 药品集中带量采购（集采，即你说的"标"）

国家/省级组织，以"承诺采购量换低价"遴选中选药品。
中选价通常大幅低于原价；公立医院须优先使用中选品种并完成约定采购量。中选结果分批进行、按省落地，并在省级药品集中采购平台挂网。这是动态、地域性极强的政策，故本表不内置集采/挂网状态。

④ 抗菌药物分级管理（卡片"抗菌分级"标签）

依《抗菌药物临床应用管理办法》(卫生部令第84号)及《2015版指导原则》。
非限制使用级(安全、低耐药、价廉，多为基药品种)·限制使用级(耐药影响较大或价格较高，需相应处方权)·特殊使用级(碳青霉烯、替加环素等末线药，须高级职称医师+会诊方可用，本表未收录)。不同级别医院可配备的抗菌药品种数受限、基层另设独立目录。分级目录由各省制定，本表标"常见分级"，务必以本省目录及本机构规定为准；抗结核/抗寄生虫/抗病毒、抗菌中药及局部应用制剂不纳入分级。

⑤ 处方权与医院分级

"几级医院用不同药"的落点。
医师按职称取得不同级别药物的处方权限(如特殊使用级抗菌药限高级职称)；医院按等级(三级/二级/基层)配备的药品广度与品种数不同。同一药在不同医院"能不能开、谁能开"可不同。
⚠ 以上均为动态、地域性政策，随版本/批次/省份变化。本表是临床用药速查，不内置实时政策状态——报销、集采、分级的确切状态，请以国家医保局及本省最新文件、本院药事会决定为准。

🧠 记忆法

核心原则：这185个药不该逐个背，该按结构记——把"散点"挂到"骨架"上。常规靠理解、能查的靠检索、力气全花在易混对和高危红线上。

① 别背能查的，背"查得快"

这张表最大的价值是把记忆负担外包。先练"检索肌肉记忆"：熟悉筛选(分类/🚑抢救车/管制/服用时机)与搜索，做到3秒定位。真正需要脱口而出的很少——抢救车那批、剂量算法、几条致命禁忌；其余知道"在表里、怎么调出来"就够。

② 背"机制"，不背"结论"

理解原理后结论自己长出来。卡片的"频次机制""理由"栏就是为这层做的。记住几条"为什么"，能覆盖几十个药：
· PPI/阿司匹林半衰期短却qd → 不可逆抑制
· 他汀有的睡前有的随便 → 半衰期长短 vs 夜间胆固醇合成
· 阿卡波糖随第一口饭 → 作用在肠道、随碳水才起效
· 肠溶阿司匹林空腹、缓释片整片吞 → 剂型决定

③ 按"药理分类成串"记，不按字母表

用页面的分类色块做记忆单元，一个类记"代表药+类规律+类禁忌"，记例外、不记常规：
· 他汀：共性(肝代谢、查肝酶肌酶、孕妇禁)，差异只在半衰期
· 二氢吡啶CCB：常规外周降压，例外=尼莫地平(脑选择性)
· 头孢按"代"记抗菌谱走向(一代偏G⁺、三代偏G⁻、抗铜绿看头孢他啶)

④ 力气全花在"易混对"和"高危红线"

泛泛的药记大概即可，会出事的细节必须钉死(用🎯自测专练)：
易混对：止血敏/止血芳酸 · 尼莫/尼群 · 茶碱/氨茶碱/喘定 · 感冒清热/感冒清 · 氨甲苯酸/氨甲环酸 · 654-2/间苯三酚
高危红线：氯化钾绝不静推 · 异丙嗪不入动脉/皮下 · 深伤口不涂油膏 · 含雄黄/乌头碱中成药 · 麻醉药品红处方双人核对 · 缓/控释整片吞

⑤ 三个具体做法

· 用就是记：开药时顺手翻它的机制栏，临床场景+主动检索，胜过干背(天然的间隔重复)。
· 自测而非重读：看药名先自己说频次/禁忌/易混点，再翻表核对，错的标记重点攻 → 用🎯自测。
· 抓异常：遇到"为什么和那个不一样"就深究，符合直觉的略过；记忆资源花在反直觉处。
一句话：常规靠理解、能查的靠检索、力气钉在易混对与高危红线。配合 🎯自测 做主动回忆 + 间隔重复，效率最高。

🧭 进阶规划 · 临床能力三柱与缺口

缘起：在把科室药品表做成本工具的过程中浮现的更大问题——"掌握急救 + 用AI掌握临床思维 + 了解手头能用的药"这三件事，拼起来够不够？以下是梳理。

① 已经做了什么

把药品表盘活成能查、能算、能练的工具：185种药的结构化字段(途径/频次+机制/服法+理由/成人·儿童·特殊人群剂量/适应症/禁忌/常用病/来源)，剂量计算器、抗菌分级、抢救车与管制标记、易混对与高危红线要点、制度与目录、记忆法、自测。
这本质上就是"第三根柱子——手头能用的药/武器库"被彻底盘活了。

② 三根柱子：能力骨架

三件事是三种性质的能力：
· 柱1 掌握急救 = 战场(高压限时的急性处理)——急诊手册+抢救车，已立
· 柱2 AI掌握临床思维 = 大脑(诊断与决策)——在建
· 柱3 了解能用的药 = 武器(可调用的手段)——本工具，已盘活
即 临床思维(脑) + 药物武器(手) + 急救场景(战场)。
完整闭环：问诊→查体→辅助检查→判读→诊断/鉴别→决策(药/操作)→监测随访→沟通记录与安全
三柱占住了思维/武器/急性场景，但柱子之间是断开的。欠的不是第四根柱子，而是连接它们的"神经"——上游读数据、中间串方案、下游看反馈。

③ 缺口一 · 上游：辅助检查的"判读"层(最大的洞)

三柱都默认"数据在手且看得懂"，但思维要有输入、急救离不开判读：
· ECG判读——抢救车的肾上腺素/阿托品/利多卡因/西地兰用不用、怎么用，全靠心电图；ECG对内科急诊是硬通货，与抢救车天然配对。
· 检验危急值/血气——补钾看血钾、地高辛/茶碱看血药浓度、华法林看INR、肾功能定剂量(肌酐清除率计算器即例)。
读不懂数据，思维和用药都悬空。优先级最高，尤其ECG。
✓ ECG 已建成 → ❤️ 心电速查（判读八步法 + 7 个致命心电图直连抢救车用药）。检验危急值/血气判读仍可继续扩（目前仅肌酐清除率计算器涉及）。

④ 缺口二 · 中间：单药 → 方案/流程

药典按"一个药"组织，临床面对的是"一个急症"：DKA、高钾血症(钙剂→葡萄糖+胰岛素→碳酸氢钠)、过敏性休克、癫痫持续状态、上消化道大出血、脓毒症bundle……这些是把多个药+剂量+时序+监测拧成一条流程。知道单药≠会处理急症。 把急症流程与药典打通，是连接柱1(急救)与柱3(药物)的桥。
✓ 已建成 → 🚨 急症处理（9 个 bundle：识别 → 药+剂量+时序 → 监测 → 红线，加粗药可点击跳主表）。

⑤ 缺口三 · 下游：开药之后

真实病人多药并用，逐药的表缺两块：
· 相互作用/多重用药——双硫仑反应、QT叠加、肾毒叠加、保钾药+ACEI致高钾。
· 用药后监测——他汀查CK/肝酶、胺碘酮查甲状腺和肺、二甲双胍查肾。
处方不是终点，随访才闭环。
✓ 已建成 → ⚗️ 相互作用·监测（9 条相互作用红线 + 11 行用药后监测表，含他汀查CK、二甲双胍查肾、胺碘酮查甲功/肺）。

⑥ 对柱2的提醒：教科书知识 ≠ 临床判断

AI能把教材结构化、鉴别诊断列全(一张完备地图)，但临床思维是在信息不全、有时间压力、有风险时做决策，只能从真实病人、反馈、不确定性里长出来。柱2的隐藏短板是缺病例复盘的闭环——而这正是CaseLog在积累的：把处理过的病人拿回来对照当时的思维看哪偏了。把CaseLog与思维训练接上，比再扫一本教材更补这块。
○ 仍开放 → CaseLog 真实病例接入"思维复盘"（本工具之外的长线整合，是三柱中唯一尚未闭合的缺口）。
结论（已更新）：连接三柱的三段神经现已全部落地——读数据 → ❤️心电速查、串方案 → 🚨急症处理、看反馈 → ⚗️相互作用·监测。三柱从"各自独立"连成了一条判读 → 选药 → 串成流程 → 用药后随访的闭环，加粗药还能一键跳回主表查详情。
唯一仍开口的是柱2的病例复盘闭环：把 CaseLog 真实病人拿回来对照当时的思维看哪偏了——这要靠本工具之外的 CaseLog 接入，是下一段长线。

❤️ 心电速查 · 判读 + 危急联动

系统读图 + 致命心电图直连抢救车用药。加粗药中带虚线者可点击跳主表。波形为示意图，定性诊断须看完整12导联并结合临床。

① 判读八步法

①节律：有无P波、P与QRS是否1:1、是否窦性(I/II/aVF直立、aVR倒置)
②心率：规则时 300÷R-R大格数 或 1500÷小格数；不规则数6秒内QRS×10
③电轴：看I与aVF主波方向(均向上=正常)
④P波：形态/时限<0.12s(增宽=左房，高尖=右房)
⑤PR间期：0.12–0.20s(延长=I度AVB；缩短+δ波=预激)
⑥QRS：<0.12s；增宽=室性/束支阻滞/预激
⑦ST-T：抬高(损伤)/压低(缺血)、T倒置
⑧QT/QTc：QTc<0.44(男)/0.46(女)；延长警惕尖端扭转
正常值：心率60–100 · PR 120–200ms · QRS<120ms · QTc<440/460ms · P<120ms · 1小格0.04s/0.1mV、1大格0.2s
示意：正常窦性 P–QRS–T

⛔ 致命心电图 → 直连处理(联动抢救车)

室颤 / 无脉性室速 (VF/pVT)
示意：杂乱无章/宽大快速
立即非同步电除颤(首要)+ CPR；肾上腺素1mg iv q3–5min；抗心律失常用胺碘酮(首选，本表未收录)或利多卡因(本表有)。
无脉电活动 / 心脏停搏 (PEA/Asystole)
CPR + 肾上腺素1mg q3–5min；查可逆病因(5H5T：低氧/低容量/低钾高钾/低体温/酸中毒；张力性气胸/心包填塞/血栓/中毒)；停搏不电除颤。
STEMI(相邻导联ST弓背向上抬高)
示意：ST弓背抬高(墓碑样)
阿司匹林300mg嚼服 + P2Y12；尽快再灌注(PCI/溶栓)；硝酸甘油缓解(下壁/右室梗死、低血压、近期用PDE5抑制剂者禁/慎)；镇痛(吗啡慎用)。
高钾血症
示意：T波高尖对称(→PR延长、P低平→QRS增宽→正弦波→停搏)
①葡萄糖酸钙稳定心肌膜(最快、不降钾) → ②葡萄糖+胰岛素、碳酸氢钠(酸中毒时)、β₂激动剂 驱钾入细胞 → ③呋塞米/树脂/透析 排钾。出现心电图改变即按急症处理，不等血钾报告。
三度房室阻滞 / 症状性心动过缓
示意：P波规律、QRS规律，二者各自独立(分离)
阿托品0.5–1mg iv(结下阻滞常无效)；经皮/经静脉起搏；备多巴胺/肾上腺素(异丙肾上腺素首选，本表未收录)。
尖端扭转型室速 (TdP)
示意：QRS振幅围绕基线渐变扭转(QT延长基础)
硫酸镁iv(首选)；纠正低钾低镁；停用一切延长QT的药；必要时超速起搏/异丙肾；不稳定即电复律。
快速心室率房颤(伴血流动力学不稳)
不稳定→同步电复律；稳定→控制室率(西地兰/β阻滞剂/非二氢吡啶CCB)。预激合并房颤禁用上述房室结阻滞药(致命)，需电复律或胺碘酮。

② 常见心律失常识别要点

· 窦速/窦缓：窦性P，>100/<60
· 房早/室早：提前的房性P波 / 提前宽大畸形无P的QRS+完全代偿间歇
· 房颤：绝对不规则、无P波、f波
· 房扑：II/III/aVF锯齿样F波，心室率常成比例(2:1等)
· 阵发性室上速(PSVT)：窄QRS、150–250次/分、规则、无明显P
· 室速：宽QRS(≥0.12)、快、规则或轻度不规则；不稳定即电复律
· 房室阻滞：I度PR>0.20固定；II度I型(文氏)PR渐长至脱落；II度II型PR固定突然脱落(危险、易进展);III度完全分离
· 束支阻滞：QRS≥0.12；RBBB(V1呈rsR'、V6宽S)/LBBB(V6宽钝R、可掩盖梗死)
⚠ 节律条只看节律，定性诊断须12导联；务必结合症状、电解质、用药史。危急心电图即刻处理，不等正式报告。波形为教学示意，非真实记录。

🚨 急症处理 · 流程速查

每个急症：识别 → 处理序列(药+剂量+时序) → 监测 → 红线。加粗药=本表收录，带虚线者可点击跳主表查详情。剂量为常用参考，以本院方案与说明书为准。
过敏性休克
识别接触变应原后速发：皮肤(荨麻疹/血管性水肿)+呼吸(喉头水肿/喘鸣)/循环(低血压)/消化道，累及≥2系统或出现低血压。
处理
  1. 立即脱离变应原、平卧抬腿、高流量吸氧、保持气道(备插管/环甲膜)
  2. 肾上腺素 0.3~0.5mg 肌注(大腿外侧中部)立即，5~15min可重复——一线、首要
  3. 快速补液：生理盐水/平衡盐晶体扩容(成人首剂500~1000ml快速)
  4. 二线辅助(不替代肾上腺素)：异丙嗪/抗组胺、地塞米松(防双相反应)；支气管痉挛→沙丁胺醇雾化/氨茶碱
  5. 顽固低血压：肾上腺素静脉泵入 / 去甲肾上腺素
监测血压、SpO₂、心电、气道；观察≥6~24h防双相反应。
⚠ 肾上腺素是首选且首要——别先用激素/抗组胺而延误；首选肌注，静脉给药仅用于循环衰竭且须稀释+监护。
心脏骤停 / 成人 ACLS
识别无反应、无呼吸或濒死叹息、无脉。
处理
  1. 立即高质量CPR(100~120次/分、深5~6cm、充分回弹、最小中断)、呼救+取除颤仪
  2. 辨节律：可除颤(VF/无脉VT)→立即电除颤后即刻CPR 2min；不可除颤(PEA/停搏)→继续CPR
  3. 肾上腺素 1mg iv/io q3~5min(不可除颤律尽早给；可除颤律第2~3次电击后给)
  4. 可除颤律电击无效：胺碘酮(首选,本表未收录)或利多卡因
  5. 高级气道、查可逆病因 5H5T；复苏后维持MAP、目标体温管理、查因(ECG→STEMI再灌注)
监测节律、ETCO₂(反映按压质量与ROSC)、每2min触脉。
⚠ 停搏/PEA不电除颤；按压质量与最小中断是核心；碳酸氢钠不常规，仅高钾/三环类中毒/严重酸中毒时用。
糖尿病酮症酸中毒 DKA
识别高血糖(常>13.9)+酮体阳性+代谢性酸中毒(pH<7.3/HCO₃⁻<18)、AG↑；多尿口渴、深大呼吸、腹痛、意识改变。
处理三支柱=补液+胰岛素+补钾
  1. 补液最优先：生理盐水首小时15~20ml/kg(约1L)，后按血钠/容量调整(高钠可换0.45%)
  2. 小剂量胰岛素(常规/短效)0.1U/kg/h 持续泵入；血糖降至13.9mmol/L时液体加葡萄糖并减胰岛素，维持至酮体清除、酸中毒纠正
  3. 补钾：见尿补钾，血钾<5.2即开始氯化钾(稀释静滴)；血钾<3.3先补钾、暂缓胰岛素(防致命低钾)
  4. 碳酸氢钠仅 pH<6.9 时谨慎使用
监测每1~2h血糖、每2~4h电解质/血气/酮体、尿量、心电(钾)。
⚠ 先(或同时)补液再胰岛素；血钾<3.3先补钾；血糖正常≠DKA纠正(看酮体/AG/酸碱);补液过快防脑水肿(尤其儿童)。
高钾血症
识别血钾>5.5；心电图 T波高尖→PR延长/P低平→QRS增宽→正弦波→停搏。
处理
  1. 心电改变或血钾>6.5立即：葡萄糖酸钙 10% 10~20ml 缓慢iv 稳定心肌膜(不降钾、最快，5min无效可重复)
  2. 驱钾入细胞：葡萄糖+胰岛素(50%葡萄糖+常规胰岛素)、碳酸氢钠(酸中毒时)、沙丁胺醇雾化
  3. 排钾：呋塞米(肾功能尚可)、阳离子交换树脂、血液透析(肾衰/顽固/危及生命)
  4. 停一切升钾药(保钾利尿剂、ACEI/ARB、补钾)
监测持续心电、每1~2h复查血钾。
⚠ 葡萄糖酸钙只护心、不降钾，必须接驱钾/排钾；洋地黄中毒所致高钾，钙剂须更慢更稀(增强洋地黄毒性)。
癫痫持续状态
识别抽搐≥5分钟，或反复发作、发作间期意识不恢复。
处理按时间阶梯
  1. 0~5min：ABC、吸氧、保持气道、测血糖(低糖→50%葡萄糖+维生素B₁)、建立静脉
  2. 5~20min 一线：地西泮 10mg 缓慢iv(≤2~5mg/min)，无效10~15min重复(总≤30mg)；或咪达唑仑肌注
  3. 20~40min 二线：丙戊酸钠/左乙拉西坦/苯妥英(本表未收录，按可得选)
  4. 40min+ 难治：麻醉药(咪达唑仑/丙泊酚)持续+气管插管+转ICU+脑电监测
监测呼吸(地西泮致抑制，备气囊面罩)、血压、血糖、找病因。
⚠ 地西泮必须缓慢推注、备气道；别只反复给苯二氮䓬而不上二线；查纠正低血糖/低钠/低钙等可逆因。
急性左心衰 / 急性肺水肿
识别突发呼吸困难、端坐呼吸、粉红泡沫痰、双肺湿啰音、SpO₂↓。
处理
  1. 端坐位、双腿下垂；高流量吸氧/无创通气(CPAP/BiPAP)
  2. 呋塞米 20~40mg iv(已用利尿剂者加量)
  3. 收缩压>90~100：硝酸甘油静脉泵入扩血管(降前负荷)
  4. 烦躁/极度呼吸困难：吗啡小剂量谨慎(呼吸抑制、低血压、COPD/意识障碍慎)
  5. 低血压/低灌注：多巴胺/正性肌力；快速房颤诱发→控室率(西地兰)
  6. 处理诱因(ACS/快速心律失常/感染/血压骤升)
监测SpO₂、血压、尿量、心电、血气。
⚠ 低血压禁用硝酸甘油；吗啡已非常规、须谨慎；务必找诱因(尤其ACS→查ECG)。
上消化道大出血
识别呕血/黑便/血便、循环不稳(心率↑、血压↓)、Hb↓。
处理
  1. 容量复苏：开放两条大静脉，生理盐水/平衡盐晶体；必要时输血(限制性目标Hb~70~80，活动性大出血/老年放宽)
  2. 抑酸：注射用奥美拉唑 80mg负荷+8mg/h持续(非静脉曲张性)
  3. 疑食管胃底静脉曲张：生长抑素/奥曲肽(本表未收录)+预防性抗生素+尽早内镜；非曲张性同样尽早内镜止血
  4. 辅助止血：必要时氨甲苯酸/酚磺乙胺(辅助、非主力)
  5. 禁食、记出入量、备血、请消化/外科
监测血压心率、尿量、Hb动态、再出血征象、意识。
⚠ 先稳循环再查因；曲张破裂需降门脉压药+抗生素，别只当普通溃疡处理。
有机磷农药中毒
识别胆碱能危象：毒蕈碱样(流涎/多汗/瞳孔缩小/支气管分泌↑/腹泻)、烟碱样(肌颤/肌无力)、中枢(意识障碍/抽搐)；胆碱酯酶活性↓、呼出大蒜味。
处理
  1. 脱离毒物、彻底洗消(脱衣、清洗皮肤)、经口者洗胃；医护防护
  2. 阿托品：反复大剂量直至"阿托品化"(口干、皮肤干、瞳孔散大、心率增快、肺啰音减少)，以体征为终点、非剂量，并维持阿托品化
  3. 胆碱酯酶复能剂：氯解磷定/碘解磷定(本表未收录)尽早、足量(对烟碱样症状/呼吸肌麻痹关键)
  4. 对症：保持气道、机械通气(呼吸肌麻痹)、抽搐用地西泮
监测阿托品化体征(防过量：谵妄/高热/抽搐)、胆碱酯酶活性、呼吸。
⚠ 阿托品看体征不看剂量；复能剂要早(磷酰化"老化"后失效)；警惕中间综合征/反跳与迟发性神经病。
脓毒症 / 感染性休克(1小时集束化)
识别感染+器官功能障碍(qSOFA：意识改变/呼吸≥22/收缩压≤100占2项);休克=充分补液后仍需升压维持MAP≥65且乳酸↑。
处理1小时内
  1. 测乳酸；用抗生素前留取血培养
  2. 尽早(1h内)经验性广谱抗生素(按感染源选)
  3. 低血压或乳酸≥4：快速晶体生理盐水/平衡盐 30ml/kg
  4. 复苏后仍低血压：去甲肾上腺素(首选升压药)维持MAP≥65
  5. 查找并控制感染源
监测MAP、乳酸清除、尿量、SpO₂。
⚠ 抗生素别等培养结果；去甲肾上腺素为首选升压(优于多巴胺)；羟乙基淀粉(706代血浆)在脓毒症禁用(增加肾损伤/死亡)。
⚠ 流程为常用参考框架，剂量/时序以本院方案、最新指南与说明书为准；危重抢救按实际团队分工与设备执行。
