// 03 本周唯一动作

把"生成代码"换成"读懂代码"

用 uni-app stars:>200 在 GitHub 找一个比你做过的更简单、但结构干净的项目，clone 下来，关掉 AI，让自己逐文件读懂它怎么组织目录、拆组件、调云函数。

为什么是这个：这一个动作，比那门课的前六章加起来更靠近 like-a-programmer。你卡的不是"会不会做产品"，是"会不会读懂别人的产品"——而读懂，就是监督 AI 这个动作本身。
