课堂讲义 · 四阶段流程

医学科研论文写作

从一个值得回答的临床问题，到一份能投出去的稿件。全程分四步：构思立题、搭建提纲、起草成文、反复修改——前一步的质量，决定下一步能走多远。

01
Conception

论文构思

立题决定论文能不能"立得住"。这一步是全流程的地基——方向错了，后面写得再好也救不回来。

凝练科学问题

  • 从临床实践或现有研究的空白中提炼问题unmet need
  • 用 PICO 框架界定：人群(P)、干预/暴露(I)、对照(C)、结局(O)
  • 判断临床意义与创新性——新人群、新方法、新结论或新机制

文献检索与定位

  • 系统检索 PubMedEmbase知网/万方，搞清这个问题"已知什么、争议在哪"
  • 找到研究缺口gap——这就是整篇论文的立论支点
  • 顺带调研同类文章都发在哪些刊，为后续选刊埋伏笔

明确研究类型与设计

  • 病例报告 / 横断面 / 队列 / 病例对照 / RCT / Meta 分析
  • 设计决定证据等级，也决定写作模板：CONSORTSTROBEPRISMA

预设核心结论与目标期刊

  • 一句话能说清"本研究证明了什么"take-home message
  • 按创新度、样本量、影响力，初步锁定 1–2 个目标刊及其 scope
02
Outline

提纲拟定

把构思转成可写的骨架。先搭结构、再填内容，避免边写边乱。

采用 IMRaD 标准结构

IIntroduction
为什么做——背景 → 缺口 → 目的，呈"倒三角"逐步收窄
MMethods
怎么做——研究设计、对象、变量定义、统计方法，按可重复性列全
RResults
发现了什么——与方法一一对应，只陈述事实
DDiscussion
意味着什么——主要发现 → 比较 → 机制 → 局限 → 结论，"正三角"展开

规划图表（最先做）

  • 先定"几张图、几张表"以及各自传达的核心信息——图表是论文的骨架
  • 结果文字围绕图表写，避免文字与图表重复描述

确认逻辑主线（闭环检查）

  • 引言提出的问题，结果是否回答？讨论是否呼应？
  • 从"问题 → 数据 → 结论"画一条线，确认中间不断裂
03
Drafting

论文起草

进入写作。原则只有一条：先成形，再求精。初稿不追求完美。

推荐写作顺序（不按阅读顺序）

方法→ 结果→ 引言→ 讨论→ 摘要→ 标题
  • 方法、结果最客观、最易写，先动笔；引言、讨论需要论证，随后
  • 摘要和标题最后写——全文定稿才能准确概括

各部分写作要点

  • 方法：过去时，细节足以让人复现
  • 结果：只陈述、不解释；配合图表给出具体数值与统计量CIP 值
  • 讨论：紧扣自己的发现做解释，避免堆砌文献；诚实交代局限性
  • 引言：简洁，3–4 段足够，直击缺口

语言与规范

  • 一段一个中心意思，主题句开头
  • 术语统一；缩写首次出现给全称
  • 边写边按目标期刊格式管理参考文献ZoteroEndNote
04
Revision

论文修改

修改往往比写作更耗时。分宏观、中观、微观三个层次反复打磨。

研究表明本药物可能也许在一定程度上显著降低了 28 天死亡率这是非常重要的发现。 — 修改 = 删冗余 · 加精度 · 去过度解读

内容层（宏观）

  • 逻辑是否自洽？结论是否被数据支撑？有无过度解读？
  • 删除冗余——确保每一段都在服务核心论点

语言层（中观）

  • 句子精简、时态一致；投英文刊须避免中式英语
  • 必要时润色，或请母语者 / 专业机构修改

格式层（微观）

  • 严格对照目标刊 Author Guidelines：字数、图表格式、参考文献样式
  • 核对图表编号、引用编号、单位与有效数字

外部反馈与终审

  • 请同行 / 导师通读，尤其让非本专业读者检验可读性
  • 放置几天后"冷读"cold reading，更容易发现问题
  • 准备 Cover Letter，预想审稿人的质疑并提前自查
诀
The Playbook

实战诀窍

老师课堂上的"私房话"——题名怎么定、名字怎么署、投稿要哪些硬件。说到底，论文也是一个人情社会。

“ 天下文章一大抄，看你会抄不会抄。 "抄"的是结构和套路，不是内容。先找一份本领域的优秀毕业论文当范本，学它的谋篇布局、行文逻辑、表达方式——这是正经的"师法范文"。但数据、观点、文字若直接搬用，就成了学术不端。会抄＝会借鉴框架，不会抄＝抄了内容。

题名：越小越好

  • 越小越好、越有指向性越好——一篇只解决一个问题
  • 偏定性表述，把研究对象和结论方向点明，别贪大求全
  • 看到题目就知道你回答了哪个具体问题，这就是好题名

署名：三类人

  • 出思想的——提出构思、设计研究的人
  • 出力气的——做实验、采集与分析数据的人 补
  • 执笔的——撰写与修改稿件的人 补

注：这与国际 ICMJE 作者署名标准（构思设计 / 数据获取分析 / 起草修订，且对终稿负责）一致。后两类是替你补的通行说法。

署名单位与通讯作者

1
第一单位：自己就读的大学（毕业论文以学位授予单位为先）
2
第二单位：自己所在的工作单位（如医院/科室）
通
导师 = 通讯作者（Corresponding Author，负责对外联络与把关）

名字千万别署错——单位、排序、通讯作者标注，投稿前逐一核对，错了很难改。

投稿硬件（少了进不去核心刊）

  • 基金项目：想投国内核心期刊基本得有；没有就不附——但课题级别越高越好，省部级以上（乃至国家级）最受认可 部分补
  • 伦理审查证明：涉及人/动物的研究须附；除非属豁免或无需伦理审查的情形，需在文中说明
  • 临床试验注册号：临床研究须提供注册登记号（如 ChiCTR / ClinicalTrials.gov），多数刊为强制要求

一句话总结

听懂了——论文，也是一个人情社会。

细
Section Rules

逐节写作细则

每一节具体怎么落笔——引言、资料与方法、结果、讨论、摘要、参考文献，各有各的硬规矩。

引言Introduction

属"经验写作"——四问展开

  • 按 为什么做 → 怎么做 → 结果 → 结果说明什么问题 的逻辑铺陈
  • 要有参考依据：梳理"一些学者认为……，是否还有另一些观点……"，在两派的交叉点 / 分歧处切入——那里就是你文章的新意所在
  • 必须有自己的观点、看法、切入点，这才叫"新文章"
  • 多引参考文献：既显文献功底，也有助于降低查重率
  • 篇幅 约 200 字；理论基础要加强
  • 常识性、公式性的东西别写——相信审稿人和评委的水平远高于你
资料与方法Materials & Methods

真实性审查，从这里查起

  • 真实、可靠、经得起推敲——查重只是"小儿科"，真正的审查从资料与方法开始
  • 入选标准：所有研究对象都要有明确的纳入/排除标准
  • 样本量要够
  • 患者满意度：若无第一手数据，可写明"由第三方提供"（第三方评价更显客观）
  • 观察指标：可单列一节，也可不单列；一般少于 5 个指标就别投了。方法和指标可以借鉴、套用——但数据一定是自己的
统计学，重中之重。必须符合统计学要求——统计学审稿人一旦退稿，文章基本就完了。
真实是底线。数据不重复、不抄袭，可溯源；网上卖的文章都是假的，远离论文工厂——编造数据是学术不端的红线。
结果Results

没有商量余地

  • 客观如实，不解释、不修饰——这一节没有商量余地
  • 数据须可溯源

图表取舍

  • 能用文字说清，就不用图表
  • 能用表，就不用图
  • 别全是黑白——必要处适当配色，但须符合期刊规范
讨论Discussion

最受重视的一节

  • 讨论体现思考——哪怕观点是错的也会被重视
  • 讲清楚应当阐明的内容
  • 提炼应得到的原理及其普遍性
  • 说明研究中有无例外、或本实验尚难解决的问题
  • 与先前公开发表的工作比较异同
  • 给出客观评价与建议
摘要Abstract

对标最新范文

  • 要投核心期刊，就去找该刊最近发表的文章当范本
  • 英文摘要与中文摘要，强调点务必一致
  • 学位论文摘要是"大摘要"（信息更全的详细摘要）
参考文献References

数量与时效

≥20论文（少了常被批"创新性不足"）
≥50论著 / 论述
≥70综述
≤3–5 年引用时效，最长不超 5 年

很多时候审稿人说"创新性不足"，其实是文献不够、没接住最新进展的信号。

由 CN 与 ISSN 共同组成 — 即 ISSN 1000-2065 ＋ CN 32-1248/R国家标准规定的完整刊号著录形式，二者缺一不可
