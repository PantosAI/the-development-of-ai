01

鲁菜（山东）

醇厚

北方菜的根，宫廷菜的底子。讲究火候足、味要醇正，咸鲜打底，最见功力的是清汤与奶汤的吊制。

⇠ 食材 海产 + 大葱 + 高汤 风味 ⇢ 咸鲜醇厚
  • 爆 / 扒旺火快攻或文火慢煨，是鲁菜两大看家技法
  • 吊汤清汤奶汤吊得清亮，「无汤不成席」
  • 糖醋鲤鱼★外焦里嫩、糖醋亮汁，黄河鲤鱼为正宗
  • 九转大肠酸甜苦辣咸五味俱全，功夫菜代表
  • 葱烧海参★大葱去腥提香，海参吸味，鲁菜门面
  • 把子肉 / 锅塌家常一面，浓油酱香、下饭实在
02

川菜（四川）

辛香

误解最深的菜系——不是只有辣。真正的精髓是「一菜一格、百菜百味」，二十多种复合味型才是底气。

⇠ 食材 花椒 + 辣椒 + 豆瓣 风味 ⇢ 麻辣 + 百味
  • 味型鱼香、怪味、麻辣、糊辣、荔枝……各成一格
  • 郫县豆瓣川菜之魂，红油与醇香的来源
  • 麻婆豆腐★麻、辣、烫、香、酥、嫩、鲜、活八字诀
  • 回锅肉★川菜第一家常菜，灯盏窝、豆瓣香
  • 水煮鱼 / 夫妻肺片麻辣的两张名片，一热一凉
  • 鱼香肉丝不用鱼却有鱼香，复合味型代表
03

粤菜（广东）

鲜爽

「不时不食」的极致追求。口味最清淡，最看重食材本味与新鲜，火候分寸拿捏冠绝各派。

⇠ 食材 海河鲜 + 时令 + 老火汤料 风味 ⇢ 清鲜本真
  • 清 / 鲜 / 嫩少调味、留本味，讲究「镬气」
  • 老火靓汤慢火煲数小时，广东人的命根子
  • 白切鸡★皮爽肉滑、蘸姜葱，最考鸡与火候
  • 烧腊（叉烧 / 烧鹅）★明炉挂烧，皮脆肉嫩、蜜香入味
  • 早茶点心虾饺、烧卖、肠粉，一盅两件慢生活
  • 清蒸海鱼豉油淋葱，只为衬出那一口鲜
04

苏菜（江苏）

本味

文人菜的代表。刀工冠绝、清鲜平和，甜咸适中，讲究造型与火候的分寸，最是雅致。

⇠ 食材 江鲜 + 时蔬 + 火腿 风味 ⇢ 清鲜雅致
  • 刀工文思豆腐切如发丝，苏菜的硬功夫
  • 炖 / 焖 / 煨文火慢功，求酥烂入味而形不散
  • 松鼠鳜鱼★花刀油炸成松鼠形，糖醋亮汁，造型菜典范
  • 清炖蟹粉狮子头★肥瘦细斩、入口即化，淮扬功夫菜
  • 盐水鸭 / 金陵烤鸭皮白肉嫩、咸香不腻，金陵风味
  • 大煮干丝豆干切丝、清汤吊鲜，淮扬早茶名物
05

闽菜（福建）

鲜爽

山海交汇的菜系。擅烹海鲜、长于制汤，「一汤十变」，口味偏清淡甜酸，红糟入菜是一绝。

⇠ 食材 海鲜 + 山珍 + 红糟 风味 ⇢ 清鲜和醇
  • 制汤「重汤」「一汤十变」，汤为闽菜灵魂
  • 红糟福建独有调味，增香添色、去腥提鲜
  • 佛跳墙★数十种山海珍味同煨，闽菜至尊
  • 荔枝肉形如荔枝、酸甜开胃的家常名菜
  • 沙茶面 / 鱼丸市井烟火，海鲜入味、Q弹鲜爽
  • 醉糟鸡红糟与酒香交融，闽味的代表手法
06

浙菜（浙江）

本味

清爽精致、不重浓油。求鲜嫩软滑、注重本味，菜名常带典故，杭帮、宁波、绍兴三路各有风骨。

⇠ 食材 湖鲜 + 时蔬 + 黄酒 风味 ⇢ 清鲜软糯
  • 炝 / 焖 / 烩手法细腻，求清而不淡、鲜而不俗
  • 黄酒入馔绍兴黄酒去腥增香，浙菜的隐味
  • 西湖醋鱼★活鱼现杀、糖醋勾芡，酸甜带蟹味
  • 东坡肉★慢火少水、酥而不烂、肥而不腻
  • 龙井虾仁茶香入虾，清雅至极的文人菜
  • 宋嫂鱼羹 / 干菜焖肉一羹一肉，见杭绍两路风味
07

湘菜（湖南）

辛香

与川菜并称两大辣派，但湘菜重「鲜辣、酸辣、香辣」，少麻多油、色浓味重，下饭是第一要义。

⇠ 食材 鲜辣椒 + 腊味 + 豆豉 风味 ⇢ 香辣酸辣
  • 辣的分别湘辣重鲜香、不靠花椒麻，与川辣分野
  • 腊味 / 烟熏湘西腊肉烟火气十足，自成一格
  • 剁椒鱼头★鲜辣红艳、鱼头嫩滑，湘菜门面
  • 辣椒炒肉★湖南人的国民下饭菜，镬气足
  • 毛氏红烧肉不放酱油、糖色红亮，肥而不腻
  • 小炒黄牛肉 / 口味虾一干锅一夜宵，鲜辣过瘾
08

徽菜（安徽）

醇厚

山区菜系，重油重色重火功。善用山珍野味与火腿吊鲜，「轻度腐败、严控火候」是它最独特的标签。

⇠ 食材 山珍 + 火腿 + 笋菌 风味 ⇢ 咸鲜浓醇
  • 烧 / 炖 / 熏重火功、讲究原汁原味、酥嫩入味
  • 火腿吊鲜徽州火腿提味，山珍因此出彩
  • 臭鳜鱼★似臭非臭、肉质蒜瓣，徽菜头号招牌
  • 毛豆腐★长出白毛再煎烧，发酵风味的经典
  • 一品锅 / 胡适一品锅层层码放、一锅成宴，徽州年节菜
  • 问政山笋火腿煨春笋，鲜到眉毛掉下来
八大菜系是怎么数出来的
  • 最早只说「四大菜系」——鲁、川、粤、苏，是各大流派的根基。
  • 清末民初扩展为「八大」，加入闽、浙、湘、徽四派，多按省份与风味流派划分。
  • 如今亦有「十大」之说，把京菜、沪菜（本帮）也算进来，但八大仍是最通行的框架。

02风味流变

八大菜系不是凭空并列的八个名字，而是一方水土养一方口味的结果。下图把「四大根基 → 八大流派」的源流，与各菜系的地理带、以及它们归属的四大风味系画在一起——看懂这张图，八大菜系就串成了一张活的版图。

辛香系（重味、下饭）
鲜爽系（求鲜、本味）
本味系（清雅、刀工）
醇厚系（重火、浓郁）
一句话读懂：八大菜系本质是「地理 × 物产 × 历史」共同塑造的口味分区。北方鲁徽靠厚味与火功扛住寒冷与山地，西南川湘以辣解决湿气与开胃，东南粤闽守着山海之鲜，江南苏浙在富庶里把「本味」做到极致。所以与其背菜名，不如先记住你的口味坐标落在哪一格——这正是下一栏「口味反查」要帮你做的事。

03口味反查

前面是「正着学」——按菜系认菜；这里是「反着查」——你心里往往先有个念头：今天想吃辣、想喝口汤、想清淡点、想点个下饭的。这一栏从你的需求出发，反向定位到菜系和具体该点哪道菜。

推荐强度说明： 闭眼必点= 几乎不会出错的招牌； 值得一试= 有特色，看个人口味； 小众尝鲜= 风味独特，先做好心理准备。
本栏为通用建议，口味因人而异——点菜前不妨问问同桌的接受度。
🌶
想要麻辣过瘾、又麻又辣
→ 02 川菜
闭眼必点
点这个水煮鱼 / 麻婆豆腐 / 毛血旺，花椒的「麻」是川菜与湘菜的最大分别。
为什么川辣讲究麻辣相济，舌头先麻后辣，层次比纯辣丰富。
小提醒怕麻就提前说「不要花椒、微辣」，川菜其实能调味型。
🔥
想要辣得鲜香、不要麻、特下饭
→ 07 湘菜
闭眼必点
点这个辣椒炒肉 / 剁椒鱼头 / 小炒黄牛肉，鲜辣不麻、镬气十足。
为什么湘菜用鲜辣椒和豆豉，主打香辣酸辣，不靠花椒，配米饭一绝。
小提醒湘菜的辣往往比想象中扎实，能吃辣再上剁椒类。
🥘
想吃个凉拌的、麻辣开胃菜
→ 02 川菜
值得一试
点这个夫妻肺片 / 口水鸡，红油凉拌，开胃醒神。
为什么川菜的凉菜自成一派，麻辣红油是它的灵魂调味。
小提醒作为一桌的头菜，先上凉菜开胃是地道吃法。
🍲
想喝一锅料足、滋补的浓汤
→ 05 闽菜
闭眼必点
点这个佛跳墙，海参鲍鱼花胶同煨，是「重汤」闽菜的巅峰。
为什么闽菜「一汤十变」，制汤是看家本领，料越足越显功夫。
小提醒正经佛跳墙价格不菲，认准用料而非只看名字。
🥣
想喝一口清润、不油腻的靓汤
→ 03 粤菜
闭眼必点
点这个老火例汤 / 炖盅，慢火数小时，清润不浊。
为什么广东人「无汤不欢」，老火汤讲究清而有味、温润养人。
小提醒老火汤喝汤为主，汤渣已无多少味，不必硬吃。
🍜
想要清汤吊出来的极致鲜
→ 01 鲁菜 / 04 苏菜
值得一试
点这个鲁菜清汤燕菜、淮扬大煮干丝，全靠吊汤的真功夫。
为什么「唱戏的腔、厨师的汤」，清汤不靠味精，全凭食材慢吊。
小提醒看似清淡，实则最考厨艺，是检验馆子水平的试金石。
🍃
想吃得清淡、尝食材本来的鲜
→ 03 粤菜
闭眼必点
点这个白切鸡 / 清蒸海鱼，少调味，全凭食材新鲜。
为什么粤菜口味最清淡，追求「不时不食」与原汁原味。
小提醒清蒸鱼最考新鲜度，鱼眼清亮、肉质蒜瓣才是好货。
🌸
想要清雅精致、带点文气的菜
→ 06 浙菜 / 04 苏菜
值得一试
点这个龙井虾仁 / 西湖醋鱼，茶香入馔、清鲜带酸甜。
为什么浙菜清爽精致，菜名多带典故，是江南文人菜的代表。
小提醒西湖醋鱼口味两极分化，先做好「酸甜带土腥」的预期。
🍵
想吃顿清爽的早茶 / 点心
→ 03 粤菜
闭眼必点
点这个虾饺 / 肠粉 / 烧卖，一盅两件，慢慢叹茶。
为什么粤式早茶是独立的饮食文化，点心精巧、清爽不腻。
小提醒虾饺看皮，半透明能见虾仁、捏出多道褶才算到位。
🍚
想点个浓油赤酱、巨下饭的
→ 07 湘菜 / 01 鲁菜
闭眼必点
点这个湘菜辣椒炒肉、鲁菜把子肉，咸香重口、配饭神器。
为什么这两派都重酱香与火候，菜本身就是为「就着白饭」设计的。
小提醒下饭菜偏咸，配一碗清汤或时蔬解腻更舒服。
🥩
想吃一块酥烂入味的红烧肉
→ 06 浙菜 / 07 湘菜
值得一试
点这个浙菜东坡肉（酱甜软糯）/ 湘菜毛氏红烧肉（不放酱油、糖色红亮）。
为什么同是红烧肉，两派调味迥异，能吃出地域口味的分野。
小提醒东坡肉偏甜，北方口味的人可先尝一块再决定加菜。
🍳
想点个不辣、人人都爱的家常菜
→ 02 川菜
闭眼必点
点这个鱼香肉丝 / 宫保鸡丁，咸甜微辣，老少皆宜。
为什么这是川菜「百味」的体现——鱼香、荔枝味型并不靠辣取胜。
小提醒点菜时强调「鱼香味、微辣」，避免端上来一盘红油。
🪷
正式宴请，想点道压得住场的大菜
→ 05 闽菜 / 01 鲁菜
闭眼必点
点这个佛跳墙（闽）或葱烧海参（鲁），都是宴席门面菜。
为什么用料贵重、功夫足、寓意好，是传统宴席的「硬菜」担当。
小提醒一桌一道大菜即可镇场，再多就显得用力过猛。
🐿
想点道好看又惊艳、适合拍照的
→ 04 苏菜
值得一试
点这个松鼠鳜鱼，花刀油炸成松鼠形，浇汁时滋滋作响。
为什么苏菜以刀工和造型见长，这道菜兼具观赏性与糖醋好味。
小提醒趁热吃，刚浇汁时外脆里嫩，凉了口感大打折扣。
🐟
想点个「闻着怪、吃着香」的特色菜
→ 08 徽菜
小众尝鲜
点这个臭鳜鱼 / 毛豆腐，似臭非臭、发酵风味独特。
为什么徽菜「轻度腌渍发酵」是一绝，闻着冲、入口却鲜香。
小提醒先问同桌接受度，发酵味劝退人，但爱的人会上瘾。

07烹饪技巧总结

这是一个会不断长大的「技法库」。每条技法都是独立、可复用的——不绑定某一道菜，而是标注它「出自哪道菜」。以后每多做一道菜、多悟一个门道，就往里加一张卡，按分类自动归位。

怎么用、怎么加：下面每张卡 = 一条技法，包含「怎么做 + 为什么 + 出自哪道菜」三件套。顶部按分类筛选；想新增时，复制任意一张卡、改五个地方即可——分类（data-cat）、技法名、怎么做、为什么、出处。筛选条会根据卡片自动生成，新分类也能直接冒出来，不用改脚本。

食材走的是斜抛物线（看那条虚线），推的时候被甩向后上方、在空中往锅柄方向飘。所以「收」不是把锅拉走，而是锅主动往后迎接落点——两者在空中碰头，食材就稳稳落回锅心。整个动作是一个闭合的椭圆圈，锅口始终在食材下方兜着。
三大翻车点
推太猛太靠前，食材飞出接应范围。→ 幅度小一点
收得太晚，锅没回来食材已落地。→ 主动迎上去接
抛太高，滞空久落点难控。→ 贴锅面翻，稳比帅重要
发力靠手腕那一勾，不是抡胳膊；先空锅练弧线，再用生米/豆子练抛接，匀速一秒一到两下。

食材预处理

肉下锅前吸干表面水

切好的肉用厨房纸把正反两面的生水都吸干再下锅。
表面带水会先把油温拉低、还四处溅油，肉只能被「煮」得发白出水，上不了色、出不了焦香。
SRC 源自 · 时蔬牛柳
食材预处理

姜去腥，葱蒜增香

腥味重的食材配姜片去腥；爆香增味靠葱末、蒜粒，别一股脑全扔进去。
姜里的姜烯酚等成分专压肉腥，葱蒜的硫化物负责出香——分工不同，下锅时机也不同。
SRC 源自 · 时蔬牛柳
食材预处理

绿叶菜先焯水，水里加盐

芦笋、绿叶菜下炒锅前先焯一下水，水里撒点盐。
焯过熟得快，不必为等它熟而把肉炒老；盐能护住叶绿素，成菜更翠、不发黄。
SRC 源自 · 时蔬牛柳
选锅·油温

不粘锅 ≈ 自带一层油

新手优先用不粘锅，几乎不会粘。
涂层的本质就是「提前铺好的油膜」，相当于替你完成了润锅这一步。
SRC 源自 · 时蔬牛柳
选锅·油温

铁锅靠「热锅凉油 + 润满整锅」防粘

铁锅先烧热再下凉油，转动锅让油布满整个锅面再下料。
高温让锅面收紧、凉油形成隔离膜，食材就不会直接焊在铁上；养好的铁锅一样不粘。
SRC 源自 · 时蔬牛柳
选锅·油温

下料前先看油温

油下锅后先判断油温（看油纹、丢点葱花试响），到位再下主料。
油温决定是「滑炒得嫩」还是「爆炒得香」；温度不够会粘锅出水，过高则瞬间焦糊。
SRC 源自 · 时蔬牛柳
火候·翻炒

荤菜先煸到半熟，先盛出来

牛柳这类肉先煎到半熟就盛出，等蔬菜快好时再回锅。
肉久炒必柴。分两段下锅，肉和菜各自都能停在最佳熟度。
SRC 源自 · 时蔬牛柳
火候·翻炒

爆炒的通用节奏：爆香→下素→回锅→尝味

①葱姜爆香 ②下蔬菜翻炒 ③肉回锅合炒 ④出锅前尝一口定咸淡与熟度。
香味要先用热油逼出来打底，难熟的先下，肉最后回锅只为合味——顺序错了味道就散。
SRC 源自 · 时蔬牛柳
火候·翻炒

怕焦就把锅端离火

下料、加调料这种「手忙」的瞬间，可以把锅端离火口几秒再操作。
离火控温比「慌着关火再开火」更稳更快，避免这几秒里食材过火发苦。
SRC 源自 · 时蔬牛柳
养锅·收尾

趁热清洁灶台、油烟机

炒完趁油渍还温热，抹布喷点洗洁精，把灶台、墙面、油烟机、餐具都抹一遍。
油渍刚溅上、没冷却凝固时一抹就掉；等它干硬，再清理要费十倍力气。
SRC 源自 · 时蔬牛柳
养锅·收尾

锅擦干 → 烘干 → 铁锅薄油养膜

锅洗后用布擦干、开小火烘干；铁锅再抹一层薄油收起来。
残水是铁锅生锈的根源；烘干后那层薄油既防锈又养出油膜，锅越用越不粘——这就是「养锅」。
SRC 源自 · 时蔬牛柳

买菜SHOPPING

彩椒切条或块，跟牛柳大小相近，受热和摆盘都协调。
葱姜蒜分三样备好：
姜片——主要用来去腥
葱末、蒜粒——爆香增味
鸡蛋备用（可用蛋清给牛柳上浆，让肉更嫩滑）。
芦笋绿色时蔬，提供脆口和颜色。
海苔纯装饰用，出锅摆盘时点缀。
备菜PREP
去皮芦笋根部老皮削掉，口感更嫩。
焯水绿叶/芦笋下锅前先焯一下：
① 后面炒的时候熟得快，不至于为等它熟把肉炒老
② 焯水时加点盐，能锁住「绿」颜色，成菜更翠
按盘摆放备好的料按下锅顺序分盘码好，炒起来手不忙脚不乱。
下锅（颠锅）STIR-FRY
选锅两种各有讲究：
不粘锅——本质上就类似提前加了一层油，省心好上手
铁锅——导热好、受热均匀，养好了一样不粘
放油倒入适量油，看/试油温，转动锅让油布满整个锅（热锅凉油、润锅，是不粘的关键）。
下料顺序①先下荤菜（牛柳），煸到半熟就先出锅，避免久炒变老。
②下葱姜末爆香，搅匀后下蔬菜翻炒；再把牛柳回锅合炒，最后尝一口看熟没熟、咸淡够不够。
控温小技下其他东西、怕一下子焦的时候，可以把锅端离火口，温度降下来再操作。

开小火把残水烘干
