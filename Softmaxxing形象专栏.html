底子是稳的——五官端正、眉形清晰、眼神不飘，这是很多人花钱也买不到的基础分。问题不在"长相"，而在增益项几乎一项都没打开：发型贴平、衣领压颈、配色单一。换句话说，你现在的状态是「素颜直出」，而可优化的杠杆全都还没拉。

  • 脸型 Shape
    圆中带方，面宽，下颌线被脂肪和短颈弱化——视觉上偏"圆润敦厚"。策略是纵向拉长、收窄两侧。
  • 肤色 Tone
    暖黄调。深色、低饱和、冷一点的颜色（深蓝/墨绿/炭灰）能压住黄气；大面积浅色会反衬显黄显宽。
  • 颈肩 Neck
    颈短、肩颈厚。圆领/高领是大忌——会把脸和脖子压成一团。开领、V 领是你的解药。
  • 当前造型 Now
    平头 + 高圆领灰毛衣，恰好把两个弱点（圆脸、短颈）同时放大。这是最该先改的组合。
    • The Method — 为什么叫 Softmaxxing

    不做硬整容，只做"温和加权"

    Looksmaxxing 里有 hardmaxxing（医美、正颌、手术，高风险高代价）和 softmaxxing（发型、穿搭、体态、护肤，低成本可逆）。你的底子足够好，不需要动刀，只需要重新分配权重。

    softmax(x)ᵢ = e^(xᵢ) / Σ e^(xⱼ)

    就像 softmax 把一组数值放大成清晰的概率分布——把已有的优势项指数级放大，把劣势项相对压低。你不是要"变成另一个人"，而是让既有的端正五官在整体里拿到更高权重。每一项增益单独看不大，叠加后输出会被拉开一个量级。

    下面三个专栏，就是按"投入产出比"从高到低排好的权重分配方案。

    Column No.3 — 发型决策（首选增益项）

    两套维度，看懂发型

    轴一 · 结构（剪法）= 骨架
    • 渐变 Fade — 两侧由短到长，位置越高对比越强、越显脸长
    • 断层 Undercut — 两侧一刀切，顶部突变长，对比最猛
    • 均匀 Crop / Buzz — 平头寸头属此类（你现在就是）
    轴二 · 顶部造型（风格）= 表情
    • 纹理烫 / 碎盖 — 蓬松空气感，韩日系主流
    • 背头 / 油头 — 向后梳起，复古商务
    • 侧分 Side Part — 清晰分线，成熟干练
    • 中分 Curtain — 少年感，挑脸型

    原理：结构决定脸型修饰，风格决定气质走向。你的圆方脸 + 面宽 + 颈短，主要靠「结构」解决——纵向拉长、两侧收窄；同时方脸忌硬挺高颅顶，顶部要「有体积但带纹理柔化」，不能直挺挺立平头。

    ★ 首选 Direction 01
    中高渐变 + 顶部纹理
    ★★★★★

    两侧中高渐变收窄面部，顶部留 4-5cm 做纹理（非立平头）。拉长脸型 + 碎感柔化方脸棱角，一举解决"圆+方"双重问题。

    修饰 拉长+柔棱角打理 低 · 发蜡随手抓成本 低
    Direction 02
    侧分油头
    ★★★★☆

    侧区渐变 + 顶部向后上方梳起，清晰侧分线增加纵向层次。气质更成熟，约会与内科医生身份都搭。

    修饰 拉长+显成熟打理 中 · 每天发泥定型成本 低
    Direction 03
    纹理烫短发
    ★★★★★

    顶区轻纹理烫 5-7cm + 两侧渐变。卷度让顶部自然蓬松、免费力打理，柔化方脸最有效。头发偏硬、懒打理首选。

    修饰 柔棱角最佳打理 极低 · 烫后免打理成本 高 · 2-3月补烫
    “两侧中高渐变推短，顶部留 4-5cm，要纹理感不要立平，整体往拉长脸型、显脸窄方向修——我脸偏圆方、脖子短。
    实战补充 · 发型师当面建议「两边和后面短一点」
    判断： 方向完全正确——这就是在给你做渐变（fade）。两侧收窄显脸窄，后颈推干净视觉上拉长脖子，正中你圆方脸 + 短颈的需求。但有两点必须当场跟他确认，否则容易翻车：
    别一刀切，要有过渡
    方脸的雷区是两侧一刀切的硬轮廓，会凸显棱角。要的是从短到长自然推的渐变，不是推成同一长度的"包菜头"。原话："两边和后面做渐变过渡，别一刀切。"
    顶部务必留够、留出纹理
    关键中的关键。两侧短了若顶部也剪短，就又变回平头、圆脸问题原样不动。强调："顶部留 4-5cm、要纹理感、别剪太短。""上多下少"的对比才是显脸长的发动机。

    ※ 若他想推到极短的高渐变（high fade）也 OK，对比更强更显脸长——前提仍是顶部留够。

    Column No.6 — 最高回报的增益项

    穿搭修现状，减重改数值

    关于司美格鲁肽：你不是"总不能吧"，而是恰好在适应症内

    中国获批人群为 BMI>30，或 BMI 27–30 合并高血压 / 高脂 / 睡眠呼吸暂停等。你 BMI 31–33，单凭一条就落在适应症内，平均可减约 17%（≈16.8kg）——正好到 80kg 目标区间。它是合规、有效的选项，不是禁忌话题。

    但它降的是食欲与胃排空，停药易反弹；恶心腹泻常见，胰腺炎史 / 甲状腺髓样癌 / MEN2 家族史为禁忌。它让"管住嘴"变容易，但底层的饮食结构与力量训练该建还得建。

    先体检，搞清代谢现状

    空腹血糖 · 糖化 · 血脂 · 肝功 · 血压 · 必要时睡眠监测。先看这个体重已造成哪些损伤——比减肥本身更要紧。

    基础盘 3 个月

    热量缺口（不极端）+ 力量训练保肌肉 + 睡眠。大基数前期掉 5–10kg 相对容易，先摘容易的果子。

    卡住或已有代谢异常 → 规范上药

    到那一步，司美是顺理成章的医疗决策。规范开方、规范随访，别碰代购仿制。

    训练怎么分 · 全身 / 上下 / 推拉腿 / 五分

    "几分法""推拉蹲"其实是同一件事的不同切法，都在回答：一周训练怎么把全身肌肉摊到不同天。原理就一句——一块肌肉练完要 48–72 小时恢复，所以拆开轮流练、轮流歇。从粗到细排成一张表：

    分法怎么分每块/周适合谁
    全身训练Full Body每次把全身大肌群都练一遍2–3 次新手 · 频率最高，最快建立习惯
    两分 · 上下肢Upper / Lower上肢(胸背肩臂)一天，下肢(腿臀)一天2 次一周练 4 次的经典选择
    三分 · 推拉腿Push / Pull / Legs按动作模式分：推 / 拉 / 蹲1–2 次进阶 · 公认性价比最高
    五分 · 部位splitBro Split一天专攻一个部位(胸/背/腿/肩/臂)1 次老手 · 追维度 · 一周能来 5 次
    "推拉蹲" = 推拉腿(PPL) · 力量训练三大基础动作模式
    推 Push所有往外推的 → 胸、肩、三头（卧推、肩推、臂屈伸）
    拉 Pull所有往回拉的 → 背、二头（下拉、划船、引体、弯举）
    蹲 Legs下肢 → 股四头、腘绳、臀（深蹲、腿举、硬拉）
    一个反直觉的关键：不是越细越高级。真正决定长不长肌肉的是"每块肌肉每周被练到几次 + 总量"，分法只是把量摊开的方式。新手用全身 / 上下，每块每周能练 2–3 次，比五分法（每块一周才 1 次）进步快得多——五分法是健美高阶玩法，新手上来就用反而慢。
    对现在的你

    大基数、目标减脂保肌肉、一周能稳定来 3 次 —— 全身训练或上下分化最划算，每块肌肉每周练到 2–3 次，护肌肉效率最高。想细一点，推拉腿(PPL)各练一次也完全可行。五分法先放着，等你瘦下来、需要更大训练量、一周能来 5 次了再考虑。

    3 个月 · 分阶段执行

    第 1 月Month 01 · 建立
    先建习惯，不求猛掉

    重点是把缺口和蛋白质吃对、把力量训练排进每周。先戒掉含糖饮料和夜宵这两个最大漏洞。目标 掉 3–4kg，更重要的是养成记录习惯（记体重、记吃了什么）。

    第 2 月Month 02 · 加速
    身体适应后稳定推进

    力量训练加重量、有氧加量。这阶段体重下降最稳，每周 0.5–1kg。开始能看到脸和下巴的变化——这正是样图里"脸清爽"的开端。目标累计 掉 7–9kg。

    第 3 月Month 03 · 验证
    看曲线，决定下一步

    若顺利，三个月 掉 8–12kg，已逼近 85kg。若明显卡平台、或体检已有代谢异常——这时候上司美格鲁肽是顺理成章的医疗决策（见 Column 06），规范开方随访。距 80kg 目标只剩最后一程。

    盯这几个指标（你是临床的，按这个查）

    指标 · 频率 · 意义
    体重每天晨起空腹固定时间称看周均值，别盯单日
    腰围每两周一次比体重更反映内脏脂肪
    空腹血糖 / 糖化启动前 + 每 3 月代谢核心指标
    血脂四项启动前 + 每 3 月甘油三酯对减重最敏感
    肝功 / 血压启动前查一次基线脂肪肝、血压随减重改善
    三个不要：不要极端断碳 / 断食（反弹 + 掉肌肉）；不要只做有氧不练力量（越减越松、基础代谢下降）；不要盯单日体重波动焦虑（水分波动 1–2kg 很正常，看周趋势）。

    一个核心：这是一项需要章法和坚持的工程，不是冲刺。先完成（把习惯立起来），再完美（抠细节）——这套你在临床上早就用熟了，迁移过来即可。

    Column No.9 — 回到安全区

    六件真正有用的事，与一条边界

    前面八栏全是具体打法。但 looksmaxxing 整个光谱里，真正值得长期保留的内核其实就这几件——它们本身就是正规皮肤科、运动医学、睡眠科学的常识。区别只在动机：为让自己舒服而做，而不是为"优化排名"。

    安全区 · 六件真正有用的事

    01 · Skincare

    护肤 · 防晒

    • 每日防晒是回报最高的一步
    • 温和清洁 + 保湿，别叠太多
    • 必要时加维 A 醇，从低浓度起
    02 · Fitness

    力量 + 减脂

    • 减脂让面部轮廓真实变清晰
    • 力量训练改善体态与线条
    • 比任何"下颌操"都更有效
    03 · Sleep

    睡眠

    • 减少水肿、改善气色与黑眼圈
    • 固定作息比补觉更重要
    • 低成本、高回报
    04 · Grooming

    发型 · 修饰

    • 合适发型是性价比最高的"换脸"
    • 胡须 / 眉毛整理，边界干净
    • 找一个真正懂的理发师
    05 · Style

    穿搭 · 合身

    • 合身 > 品牌 > 数量
    • 建立一套简单可复用的基础色系
    • 姿态与体态本身就是穿搭
    06 · Dental

    牙齿 · 正畸

    • 正规正畸改善咬合与笑容
    • 洁牙、牙齿健康是基础
    • 由牙医评估，不靠舌头顶上颚
    边界线 · 它在哪里变味

    同样这六件事，一旦动机从"让自己舒服"变成"提升 SMV、优化排名"，语言里开始冒出 mogging / blackpill / face card 这类词——就已经滑出安全区了。那套话语会把自我照顾扭成一场针对"排名"的军备竞赛，越陷越焦虑、越比越没底。

    而 Softmaxxing 真正的价值正在于：它不需要那套话语也成立。护肤、健身、睡眠、修饰、穿搭、牙齿——把它们当成正常的自我照顾就够了，证据支撑、无侵入、无伪科学，做了就是赚的。
